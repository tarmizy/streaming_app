# App Icon - TOY

## Design Specs
- Size: 1024x1024 pixels
- Background: Gradient biru ke pink
  - Top-left: #8B7CF7 (Light purple blue)
  - Center: #6B5CE7 (Purple blue)
  - Bottom-right: #E966A0 (Pink)
- Text: "TOY" warna putih, bold, centered
- Corner radius: ~22%

## Colors
- Primary: #6B5CE7 (Purple blue)
- Secondary: #8B7CF7 (Light purple)
- Accent: #E966A0 (Pink)
- Light Pink: #F49AC1

## Files needed
1. `app_icon.png` - 1024x1024 main icon
2. `app_icon_foreground.png` - 1024x1024 transparent background

## Generate
```bash
flutter pub get
dart run flutter_launcher_icons
```
